# 개발 명령어 및 배포 가이드

## 전체 개발 흐름

```text
VS Code에서 수정
      ↓
Ctrl + S 저장
      ↓
로컬 실행 확인
      ↓
git add .
      ↓
git commit -m "작업내용"
      ↓
git push
      ↓
Render 자동 배포
      ↓
운영 URL 확인
```

## 1. 로컬 개발 서버 실행

```bash
npm run dev
```

브라우저에서 확인합니다.

```text
http://localhost:3000
```

## 2. 빌드 확인

```bash
npm run build
```

배포 전에 이 명령이 성공하면 Render 배포 실패 가능성이 줄어듭니다.

## 3. Git 상태 확인

```bash
git status
```

## 4. 변경 파일 추가

```bash
git add .
```

## 5. Commit 생성

```bash
git commit -m "작업내용"
```

예시:

```bash
git commit -m "docs 프로젝트 관리 문서 추가"
```

## 6. GitHub 업로드

```bash
git push
```

## 7. Render 배포

`git push` 이후 Render가 자동으로 Build와 Deploy를 실행합니다.

운영 주소:

```text
https://ruahnote-bp6m.onrender.com
```
