# 프로젝트 폴더 구조

## 추천 구조

```text
ruah-note/
├── docs/
├── public/
├── src/
│   ├── app/
│   ├── components/
│   ├── hooks/
│   ├── lib/
│   ├── services/
│   ├── store/
│   ├── styles/
│   ├── types/
│   └── utils/
├── supabase/
├── scripts/
├── tests/
├── .env.local
├── .env.example
├── package.json
├── README.md
└── next.config.ts
```

## 폴더 역할

| 폴더 | 역할 |
|---|---|
| `src/app` | Next.js App Router |
| `src/components` | UI 컴포넌트 |
| `src/hooks` | React Hook |
| `src/lib` | 공통 라이브러리 |
| `src/services` | 외부 API/DB 서비스 |
| `src/types` | TypeScript 타입 |
| `src/utils` | 유틸 함수 |
| `docs` | 프로젝트 문서 |
| `public` | 정적 파일 |
| `supabase` | DB migration/seed |
| `scripts` | 자동화 스크립트 |
| `tests` | 테스트 코드 |
