# OpenAI API 설정

## 목적

강의노트 요약, 키워드 추출, 과제 후보 추출, STT 기능을 위해 OpenAI API를 연결합니다.

## 예정 기능

- [ ] 텍스트 요약
- [ ] 강의노트 생성
- [ ] 키워드 추출
- [ ] 과제 후보 추출
- [ ] STT 음성 변환
- [ ] 사용량 제한
- [ ] 비용 관리

## 예상 환경변수

```env
OPENAI_API_KEY=
```

## 주의사항

API Key는 클라이언트 컴포넌트에 직접 넣으면 안 됩니다.  
반드시 서버 API Route 또는 Server Action을 통해 호출합니다.
