# RuahNote v1.0 Project Index

## 1. 프로젝트 개요

**RuahNote**는 강의, 과제, 자료, 녹음, AI 요약, 일정 연동을 하나로 관리하는 학습·업무 노트 시스템입니다.

## 3. v1.0 핵심 목표

RuahNote v1.0은 다음 기능을 포함하는 1차 실사용 버전을 목표로 합니다.

1. 사용자 로그인
2. 카테고리/학기/과목/프로젝트 관리
3. 날짜별 노트 작성
4. 자료 첨부
5. 과제 관리
6. AI 요약 및 강의노트 생성
7. 기본 검색
8. Render 배포 운영
9. 운영 문서화
